Number Formatter
=========

A small library that adds commas to numbers

## Installation

  `npm i number-formatter-ortizmj12`

## Usage

    var numFormatter = require('number-formatter-ortizmj12');

    var formattedNum = numFormatter(35666);
  
  
  Output should be `35,666`


## Tests
